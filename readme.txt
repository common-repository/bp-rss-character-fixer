== Description ==

Fixes Buddypress's RSS feed titles' corrupted characters for all languages. 

== Installation ==

1. Upload `bp-rss-character-fixer` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Check your RSS feed (e.g. example.com/activity/feed)
